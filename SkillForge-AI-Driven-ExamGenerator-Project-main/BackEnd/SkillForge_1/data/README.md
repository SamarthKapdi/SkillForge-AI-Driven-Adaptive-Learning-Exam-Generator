# Database Folder

This folder contains the **H2 database file** (`skillforge.mv.db`) that stores all application data.

## ⚠️ IMPORTANT - DO NOT DELETE!

This folder contains:
- User accounts (login credentials)
- Courses created by instructors
- Quizzes and quiz attempts
- Student progress and analytics
- All other application data

## 📦 Sharing with Teammates

When you zip the project folder to share with teammates, **this folder will be included**.
Your teammate will see all the data/progress you created - they can continue from where you left off!

## 🔄 Fresh Start

If you want to start fresh with an empty database:
1. Stop the backend server
2. Delete `skillforge.mv.db` file
3. Restart the backend - a new empty database will be created
